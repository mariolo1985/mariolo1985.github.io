Github Setup

  1. Create an account on Github
  2. Download the Git command line tool
    • Click Here For Download Page
  3. Open the terminal on MAC or Run 'cmd' on Windows

  4. *First Time setup*

Repository Setup

  1. Create a new folder on your local machine
  2. Navigate to the folder you created earlier
  3. Create a new repository on github's webpage
  4. Clone the repository
    • Go to your repository on the webpage
    • Click the button on the page
    • Copy the URL
    • Inside the terminal type the following command and hit ENTER
    • A new folder with your repo will be created and you are now ready for versioning.

Committing Changes

  1. Open the terminal on MAC or Run 'cmd' on Windows
  2. Run a Git Status to verify changes
  3. Add your files for commit. The file name is case-sensitive.
  4. Add a commit message. Usually a helpful note on what was updated
  5. You can also use the text editor in MAC terminal
  6. Push your changes to git
    • You can also commit and push at the same time
  7. Check your repo on the webpage to ensure your changes were committed successfully
